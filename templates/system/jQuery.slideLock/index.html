slideLock Demo

Use the form below to try out slideLock. The plugin will insert the slide ui element before the submit button on whichever form it is attached to. The labels, icon and test values are all customizable.

Note that a hidden field is inserted as well and carries the test value for which you should check on the server side.

Obviously this plugin will fail without Javascript and therefore server side validation must take this into account.
