Order Status History
Reflects the statuses that an order went through on Amazon and corresponding status changes in your website. The last row in this table reflects the latest update on this order.

The order status history is updated when
  • An order moves into a new state on Amazon
  • A request is sent from OSCommerce orders UI to Amazon in order to update the status.

The order status history shows the following information:
StatusExplanation
Amazon Order Status Reflects the operation performed on this order at amazon (Unshipped\Shipped\Refund Applied\Canceled) with a corresponding status (Initiated\Succes\Failure)
OSCommerce Order Status Reflects the corresponding status applied to this order on your website.
Initiated On The time when the operation was initiated from your website requesting Amazon for a status change on the order
Completed On The time when a notification was received by your Website after the operation completed on Amazon.
Comments You can use this message to decide the next action for this order.
